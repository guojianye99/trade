# 量化交易策略回测系统

一个基于 Python 的量化交易策略回测框架，支持多种技术指标策略，包含数据管理、回测引擎和交易信号分析功能。

## 主要特性

- ✅ **多策略支持**: Dual Thrust、均线交叉、MACD、RSI、布林带、KDJ 等多种技术指标策略
- ✅ **智能建议系统**: 根据用户实际持仓和成本价，提供针对性的买卖建议
- ✅ **盈亏分析**: 自动计算持仓盈亏金额和比例，给出合理的止盈止损建议
- ✅ **分红处理**: 自动计算分红收益，提供更准确的回测结果
- ✅ **数据管理**: 支持从 baostock 下载 A 股数据，本地缓存管理
- ✅ **完整测试**: 包含 41 个单元测试用例，确保代码质量

## 策略说明

### Dual Thrust 策略
- 基于过去 20 日的价格区间计算上下轨
- 价格突破上轨买入，跌破下轨卖出
- 参数 `k` 控制突破敏感度（默认 0.5）

### 均线交叉策略
- 短期均线上穿长期均线（金叉）买入
- 短期均线下穿长期均线（死叉）卖出
- 默认使用 MA5 和 MA20

### MACD 策略
- MACD 线上穿信号线买入
- MACD 线下穿信号线卖出
- 默认参数：快线 12，慢线 26，信号线 9

### RSI 策略
- RSI 从超卖区（<30）上穿买入
- RSI 从超买区（>70）下穿卖出

### 布林带策略
- 价格从下轨反弹买入
- 价格从上轨回落卖出
- 默认周期 20，标准差倍数 2.0

### KDJ 策略
- 超卖区（K<30）金叉买入
- 超买区（K>70）死叉卖出

## 注意事项

1. **数据来源**: 本系统使用 baostock 获取 A 股历史数据，数据可能存在延迟
2. **回测局限**: 历史表现不代表未来收益，策略仅供参考
3. **风险提示**: 股市有风险，投资需谨慎。本系统不构成投资建议
4. **数据更新**: 建议定期运行 `python download_data.py --update` 更新数据
5. **信号强度**: 信号强度越高，表示策略对该信号越有信心
6. **多策略验证**: 建议综合多个策略的信号进行决策

## 常见问题

**Q: 为什么有些策略显示"观望"？**  
A: 当价格未触及关键点位或技术指标未达到触发条件时，策略会建议观望，等待更明确的信号。

**Q: 不同策略给出的建议不一致怎么办？**  
A: 建议关注"最佳策略"（历史收益最高的策略）的信号，同时参考信号统计，多数策略一致的信号更可靠。

**Q: 止盈止损建议是如何计算的？**  
A: 系统根据您的持仓成本和当前盈亏比例，动态计算合理的止盈止损价位：
- 盈利 ≥15%: 建议减仓或移动止盈
- 盈利 <10%: 建议设置保本止损
- 亏损 5-10%: 设置止损线在成本价下方 8-10%
- 亏损 ≥10%: 建议考虑止损离场

**Q: 如何判断信号是否可靠？**  
A: 可以参考以下几点：
1. 信号强度（confidence）越高越可靠
2. 多个策略发出相同信号
3. 趋势强度为"强"的信号
4. 历史收益高的策略发出的信号

**Q: 数据下载失败怎么办？**  
A: 
1. 检查网络连接
2. 确认股票代码格式正确
3. 尝试使用 `--force` 参数强制重新下载
